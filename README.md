### ***Sync_Code***
---
#### *Sync Code is a web platform where you can code syncronously in a room with other members*

---

#### ***Technologies used:***
> ***Frontend***
* ***ReactJs***
* ***Socket.io-client as Web-socket client***

> ***Backend***
* ***NodeJs***
* ***ExpressJs***
* ***MongoDB***
* ***Socket.io for Web-socket communication***
